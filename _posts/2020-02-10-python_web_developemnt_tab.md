# 개발자 도구
구글 크롬은 자체적으로 웹 개발자를 위한 개발자 도구를 제공하고 있다. 구글 크롬 개발자 도구는 **개발자들을 위하여 구글에서 제공하고 있는 개발 도구** 이다. 대부분의 웹 브라우저에서 지원하고 있는 기능인데 이 중 구글 크롬의 개발자 도구가 가장 속도가 빠르고 기능도 뛰어나다.

### 개발자 도구 shortcut
1. F12
2. Ctrl + Shift + I (windows)
3. Cmd + Opt + I (Mac)
