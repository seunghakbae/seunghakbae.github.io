[2437문제](https://www.acmicpc.net/problem/2437)
# 파이썬 코드
~~~python
N, A = int(input()), sorted(list(map(int, input().split())))

# 만들 수 있는 최저
ans = 0

print(A)


for i in A:
    if i <= ans + 1:
        print(ans)
        ans += i
    else:
        break

print(ans + 1)
~~~
